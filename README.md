# PRODIGY_WD_05 - Weather Web Application

## 📌 Description
This project was developed as part of the Prodigy InfoTech Web Development Internship.

The Weather Web Application is built using HTML, CSS, and JavaScript. It provides real-time weather information using the Open-Meteo API, allowing users to search for any city and view the current weather along with a 5-day weather forecast through a clean and responsive user interface.

## ✨ Features
- Search Weather by City
- Real-Time Weather Information
- 5-Day Weather Forecast
- Weather Icons
- Dynamic Background Based on Weather
- Temperature Display
- Humidity Information
- Wind Speed Display
- Loading Indicator
- Enter Key Search Support
- Responsive Design

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- Open-Meteo API

## 📂 Project Structure
PRODIGY_WD_05/
│── index.html
│── style.css
│── script.js


## 🚀 How to Run
1. Download or clone this repository.
2. Open the project folder in Visual Studio Code.
3. Open `index.html` in your browser or use the Live Server extension.
4. Enter a city name and click **Search** (or press **Enter**) to view the weather.
